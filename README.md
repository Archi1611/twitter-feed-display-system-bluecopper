Twitter Feed Display System with QR Code

This project displays tweets sequentially on a public screen along with
dynamically generated QR codes that link to the original tweet.

## Features
- Backend-controlled tweet rotation
- QR code generation for each tweet
- Flicker-free display updates
- Multi-handle mock support
- Public display optimized UI

## Tech Stack
- Node.js
- Express.js
- HTML, CSS, JavaScript
- QRCode.js

## How to Run

1. Install dependencies:
npm install

2.Start the server:
node server.js

3.Open browser at:
http://localhost:3000

Notes

Mock tweet data is used due to Twitter API rate limits.
The system is API-ready and can be extended for live data.
